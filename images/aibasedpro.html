AI Based Mini Project

AI Website Builder Using Speech Recognition

Objective: To design a system that transforms Tamil voice input into functional websites using AI, enabling effortless website creation. The goal is to provide real-time preview and download options, making development accessible for all users.

Technologies Used:
Frontend: HTML, CSS, JavaScript
Backend: Node.js with Express.js
AI Integration: OpenAI GPT API
Speech Processing: Web Speech API (Tamil Speech-to-Text)
Deployment: GitHub + Microsoft Azure Web App

Description: The system allows users to create complete websites through Tamil voice commands. Spoken inputs are recognized, translated, and interpreted by an AI model to generate responsive website code automatically. The platform also offers instant preview and a download option, making website creation fast and accessible without manual coding.
